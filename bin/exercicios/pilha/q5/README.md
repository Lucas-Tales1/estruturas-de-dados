# Questão 5

Projete uma classe que contenha duas pilhas, uma “vermelha” e outra “preta” e suas operações são adaptações
“coloridas” das operações habituais sobre pilhas. Por exemplo, esta classe deve prover uma operação de push
vermelha e uma operação de push preta. Usando um único array cuja a capacidade é limitada por um tamanho N
que é sempre maior do que os tamanhos somados das duas pilhas. (P-42). A pilha “vermelha” pode começar no
início do array e a pilha “preta” pode começar no final do array.
