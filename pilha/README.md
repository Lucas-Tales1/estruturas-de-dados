# Pilha

<img src="../images/stack.png" alt="pilha" width="400">

Pilha é um TAD(Tipo Abstrato de Dado) que armazena objetos arbitrários com inserções e remoções seguindo o esquema LIFO(Last in, First Off).

## Principais operações

- push(Object element) //insere um elemento

- pop() //remove e retorna o último elemento inserido

## Operações auxiliares

- top() //retorna o último elemento inserido

- size() //retorna o número de elementos armazenados 

- isEmpty() //indica se há ou não elementos na pilha
